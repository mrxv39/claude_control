Claudio Control

Monitor visual de sesiones Claude Code para Windows Terminal

Un chip por cada sesion de Claude, con colores segun su estado

Click para enfocar

Requisitos
✓ Windows 10 u 11
✓ Windows Terminal (viene preinstalado en Win 11)
✓ Claude Code CLI instalado y funcionando
Instalacion (5 minutos)
1 Descarga el .exe

No necesitas instalar nada mas. Es un archivo portable.

⬇ Descargar ClaudioControl.exe
2 Ponlo en una carpeta

Crea una carpeta donde quieras y mete ahi el .exe. Por ejemplo:

C:\ClaudioControl\ClaudioControl.exe
3 Ejecuta el .exe

Haz doble click en ClaudioControl.exe. Te aparecera una barrita en la parte superior de la pantalla.

💡 Windows puede mostrar una alerta

Si te sale "Windows protegió tu PC", pulsa "Mas informacion" y luego "Ejecutar de todos modos". Es normal con apps sin firmar.

4 Configura el hook (solo una vez)

Esto le dice a Claude Code que avise a la app cuando empieza o termina de trabajar. Abre PowerShell y pega este comando:

& "C:\ClaudioControl\resources\setup-hook.ps1"

Cambia la ruta si pusiste el .exe en otro sitio. Te saldra un mensaje verde de confirmacion.

⚠ Si te da error de permisos

Ejecuta primero esto en PowerShell (solo una vez):

Set-ExecutionPolicy RemoteSigned -Scope CurrentUser
5 Listo!

Abre sesiones de Claude Code en Windows Terminal y veraslas aparecer como chips en la barra.

Como se usa
Click normal en un chip

Trae la ventana de esa sesion al frente. Si el proyecto tiene un servicio corriendo (ej: un servidor de desarrollo), veras un badge debajo — haz click en el badge para traer esa ventana.

Ctrl + Click para organizar ventanas en mosaico

Manten pulsada la tecla Ctrl y haz click en los chips de las sesiones que quieras ver a la vez. Se seleccionaran (borde azul) y las ventanas se colocaran automaticamente en mosaico:

2 ventanas → lado a lado
3 ventanas → tres columnas
4 ventanas → cuadricula 2x2
5 o mas → cuadricula automatica

Para quitar una ventana de la seleccion, Ctrl+Click otra vez en su chip. Para limpiar la seleccion, haz click normal (sin Ctrl) en cualquier chip.

Arrastra la barra

La barra empieza centrada arriba, pero puedes arrastrarla donde quieras y recordara la posicion.
